# h_carousel.js

   **description**
    vertically-smoothly-slider-carousel

   **Usage:**
   
       var a = new h_carousel({
           $container:$("ul"),
           
           ctrl:{
               prev:$(".prev"),
               next:$(".next")
           },
           autoPlay:false,
           delay:3000,
           direction:0,
           hoverPause:false
       })
